# Read Me First
This is a demo project to play around with [Spring Cloud Gateway](https://docs.spring.io/spring-cloud-gateway/docs/current/reference/html/#gateway-how-it-works). The demo forwards requests to [HTTPBin](https://httpbin.org/) as a backend.

The file `rest-api.http` includes some requests relevant for the demo.
